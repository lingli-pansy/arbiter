## Recommendations

### For Production Use

1. **Preferred Integration Pattern:**
   ```python
   # 1. Get native data
   native = get_market_bars_batch(symbols, lookback_days, timeframe)
   
   # 2. Get venue mapping
   venues = get_symbol_venue(native["data"].keys())
   
   # 3. Convert to NT format
   nt_bars = convert_bars_to_nt(native, venues, timeframe)
   
   # 4. Feed to NT backtest
   backtest.run(nt_bars)
   ```

2. **Venue Mapping Caching:**
   - Symbol → Venue 映射相对静态，建议缓存
   - 减少重复查询开销

3. **Error Handling:**
   - 当前 venue 解析失败时会回退到 "SMART"
   - 建议记录并监控 venue 解析失败率

# TICKET_20250314_001_FOLLOWUP_003

## Source Task
VALIDATION_20250314_001_ARCH_FIX - IB 市值数据获取失败

## Blocking Issue
IB Paper Trading 账户不支持 Fundamental Data API，返回错误 10358:
```
Error 10358: Fundamentals data is not allowed.
```

## Root Cause
IB API `reqFundamentalData` 需要:
- 真实资金账户 (Live Account)，或
- 特定数据订阅 (如 Reuters Fundamental)

Paper Trading 账户默认不包含基本面数据权限。

## Proposed Solutions

### 方案 A: 使用 IB Live 账户测试 (推荐长期)
- 使用真实 IB 账户连接 (port 7496/4001)
- 需要账户有 Fundamental Data 订阅
- 可能需要额外数据费用

### 方案 B: 替代 API 获取市值 (推荐短期)
使用 IB `reqMktData` 的 generic tick 165 (Misc. Stats):
```python
# tick 165 包含市值信息
ib.reqMktData(contract, genericTickList='165')
```

或尝试 `reqContractDetails`:
```python
details = ib.reqContractDetails(contract)
# 检查 details[0].contract 或底层属性
```

### 方案 C: 估算市值 (无需额外 API)
使用 `get_market_bars_batch` 获取价格 + 预设流通股数:
```python
# 预定义大盘股流通股数（定期更新）
SHARES_OUTSTANDING = {
    "AAPL": 15.7e9,  # 157亿股
    "MSFT": 7.4e9,
    "NVDA": 24.6e9,
    ...
}
market_cap = price * SHARES_OUTSTANDING[symbol]
```

### 方案 D: 切换数据源 (架构妥协)
暂时使用 Yahoo/Polygon 获取市值，但标记为临时方案。

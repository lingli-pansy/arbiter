### 主要错误码

- **10089**: "Requested market data requires additional subscription for API" — 实时 generic ticks 需付费订阅  
- **321**: "Snapshot market data subscription is not applicable to generic ticks" — Snapshot 不适用于 Tick 165  
- **SNAPSHOT_NO_API** (Gateway 日志): 当前账户无 API Snapshot 权限  
- **Historical Data Farm: Inactive**: 延时/历史数据服务未激活

### 延时数据调研

- Error 10089 提示 "Delayed market data is available"，代码已尝试 `reqMarketDataType(3)` 请求延时数据  
- Gateway 日志显示 Historical Data Farm 为 Inactive，延时数据当前不可用  
- Generic tick 165 是否包含在免费延时包内未在 IB 文档明确，实测未成功

### 订阅与费用结论

- 用户已有 TWS 订阅「美国非整合实时报价」，但对 **API/Non-Display** 需单独开通  
- Gateway 蓝链 "Click here to subscribe to API-specific market data packages" 指向 API 专用包  
- **US Equity and Options Add-On Streaming Bundle (NP)** 可覆盖 generic ticks，约 4.50 USD/月  
- 前置条件：需先订阅 US Securities Snapshot and Futures Value Bundle (NP)

### 建议决策（供架构师）

1. **短期**: 继续使用 `source=static`，MVP 已验证可用  
2. **中期**: 若需盘中实时市值，评估付费 Streaming Bundle 成本效益  
3. **长期**: 可考虑 Polygon、Yahoo 等备选数据源作为扩展
